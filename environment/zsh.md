# ZSH - Oh My ZSH

ZSH is an awesome Unix shell Unix. Oh My ZSH is a framework to manage ZSH configuration.

## Usefull plugins
```
git
colored-man-pages
history-search-multi-word
zsh-autosuggestions
zsh-syntax-highlighting
zsh-you-should-use
zsh-bat
```

## Usefull aliases

### Custom aliases
```shell
alias zshconfig="nano ~/.zshrc"
alias reload="source ~/.zshrc"
```
